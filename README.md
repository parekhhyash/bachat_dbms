# Bachat - Expenditure Manager (Tailwind + Figtree)

## Notes
- Tailwind is loaded via CDN in base.html.
- Figtree font is imported via the Google Fonts @import you provided.
- The project uses SQLite; `create_db.py` initializes the DB.

## Run
1. (optional) venv
2. pip install -r requirements.txt
3. python create_db.py
4. python app.py
5. Open http://127.0.0.1:5000/
